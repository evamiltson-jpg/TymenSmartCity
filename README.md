# Тюмень - Smart City

Это веб-приложение является демонстрационной страницей инициативы "Умный город Тюмень". Проект демонстрирует новости, события, проекты и полезные ресурсы, связанные с цифровизацией города.

## Технологии

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Vite**
- **pnpm**

## Запуск проекта локально

Для запуска проекта на вашем компьютере выполните следующие шаги:

1.  **Клонируйте репозиторий (если еще не сделали):**
    ```bash
    git clone https://github.com/Zqqqbs/tyumen-smart-city.git
    cd tyumen-smart-city
    ```

2.  **Установите pnpm (если он не установлен):**
    ```bash
    npm install -g pnpm
    ```

3.  **Установите зависимости:**
    ```bash
    pnpm install
    ```

4.  **Запустите сервер для разработки:**
    ```bash
    pnpm dev
    ```
    После этого откройте адрес, указанный в терминале (обычно `http://localhost:5173`), в вашем браузере.

## Развертывание

Проект автоматически собирается и развертывается на GitHub Pages при каждом пуше в ветку `main` с помощью GitHub Actions.
